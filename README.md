# 📘 Supervised Learning for Medical Diagnosis

## 🔍 Project Overview
This project presents a comparative analysis of supervised learning models applied to the Breast Cancer dataset.

The study compares:
- Logistic Regression
- Support Vector Machines (Linear, Polynomial, and RBF kernels)

The aim is to understand how model complexity and kernel selection affect classification performance in medical diagnosis.

---

## 📊 Dataset
The dataset used is the Breast Cancer Wisconsin dataset, containing 569 samples with 30 numerical features.

Target classes:
- Malignant (1)
- Benign (0)

---

## ⚙️ Models Used
- Logistic Regression
- Linear SVM
- Polynomial SVM
- RBF SVM

---

## 📈 Results Summary
- RBF SVM achieved the best performance
- Logistic Regression provided strong baseline results
- Polynomial SVM showed signs of overfitting
